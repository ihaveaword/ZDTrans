ZDTrans v1.2.3 - 智能翻译助手
================================

感谢使用 ZDTrans！

安装说明
--------
1. 将 ZDTrans.app 拖拽到"应用程序"文件夹
2. 首次运行时，右键点击 ZDTrans.app，选择"打开"
3. 在弹出的安全提示中，点击"打开"
4. 授予辅助功能权限（系统偏好设置 → 安全性与隐私 → 辅助功能）

主要功能
--------
✨ 全局快捷键翻译（Ctrl+Q）
✨ 文本润色（Ctrl+Shift+Q）
✨ 日间/夜间主题切换
✨ 自定义学科领域
✨ 专业术语关键词优化

快速开始
--------
1. 首次运行需要配置 API Key
2. 选择任意文本，按 Ctrl+Q 快速翻译
3. 点击主窗口左下角的月亮图标切换夜间模式

支持的 API
