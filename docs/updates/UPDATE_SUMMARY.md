# 功能更新总结 - v1.2.0

## ✨ 新增功能

### 🌙 1. 夜色模式
- **功能**：支持日间/夜间主题切换
- **入口**：主窗口左下角 🌙/☀️ 按钮
- **特点**：
  - 所有窗口实时同步
  - 护眼深色设计
  - 设置自动保存
  
### 🎓 2. 自定义学科领域
- **功能**：可添加任意学科领域
- **入口**：设置 → 翻译配置 → 学科领域（可编辑）
- **特点**：
  - 不受限于9个预设领域
  - 一键添加到列表（`+` 按钮）
  - 配置持久化保存

## 📝 使用方法

### 夜色模式
```
方式1: 点击主窗口左下角 🌙 按钮
方式2: 设置 → 界面配置 → 主题 → 夜间模式
方式3: 配置文件 "ui": {"theme": "dark"}
```

### 自定义学科领域
```
方式1: 设置 → 翻译配置 → 直接输入领域名 → 点击 + 按钮
方式2: 配置文件 "translation": {
         "domain": "量子计算",
         "custom_domains": ["量子计算", "区块链技术"]
       }
```
