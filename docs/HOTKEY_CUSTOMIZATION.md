# 快捷键自定义功能使用指南

## 📌 功能概述

ZDTrans 现已支持自定义全局快捷键！您可以根据个人习惯设置翻译和润色的快捷键组合。

## 🎯 如何设置快捷键

### 方法一：通过设置界面（推荐）

1. **打开设置界面**
   - 右键点击系统托盘图标
   - 选择"设置"菜单项

2. **录制快捷键**
   - 在"快捷键配置"区域，找到"翻译快捷键"或"润色快捷键"
   - 点击对应的"录制"按钮
   - 按下您想要的快捷键组合（例如：`Ctrl+T`）
   - 快捷键会自动识别并显示

3. **保存设置**
   - 点击"保存"按钮
   - 新的快捷键立即生效，无需重启程序

### 方法二：编辑配置文件

如果您更喜欢直接编辑配置文件：

1. 打开 `config.json` 文件
2. 找到 `hotkey` 配置节：

```json
{
  "hotkey": {
    "translate": "Ctrl+Q",
    "polish": "Ctrl+Shift+Q"
  }
}
```

3. 修改为您想要的快捷键
4. 保存文件并重启程序

## ⌨️ 快捷键格式说明

### 支持的修饰键

- `Ctrl` - Control 键
- `Shift` - Shift 键
- `Alt` - Alt 键（macOS 上的 Option 键）
- `Cmd` - Command 键（仅 macOS）

### 快捷键格式

- 使用 `+` 连接多个按键
- 修饰键在前，普通键在后
- 不区分大小写

### 有效示例

✅ `Ctrl+Q`
✅ `Ctrl+Shift+T`
✅ `Alt+X`
✅ `Ctrl+Alt+P`
✅ `Cmd+E` (macOS)

### 无效示例

❌ `Q` - 必须包含至少一个修饰键
❌ `Ctrl` - 必须包含至少一个普通键
❌ `Ctrl++` - 格式错误

## 🔧 快捷键录制器功能

### 特点

- **即时反馈**：按下按键时立即显示组合
- **自动验证**：确保包含修饰键
- **冲突检测**：（未来版本将添加）
- **一键清除**：快速清空设置

### 使用技巧

1. **录制时**
   - 同时按下所有想要的键
   - 释放任意键完成录制
   - 支持多个修饰键组合

2. **取消录制**
   - 点击"按下快捷键..."按钮
   - 或使用 ESC 键（未来版本）

3. **清除快捷键**
   - 点击"清除"按钮
   - 将恢复默认值

## 🎨 推荐快捷键组合

根据使用场景，这里有一些推荐的快捷键设置：

### 学术场景
- 翻译：`Ctrl+T` (Translate)
- 润色：`Ctrl+P` (Polish)

### 开发场景
- 翻译：`Ctrl+Shift+T`
- 润色：`Ctrl+Shift+P`

### 阅读场景
- 翻译：`Ctrl+Q` (Quick)
- 润色：`Ctrl+E` (Enhance)

### macOS 用户
- 翻译：`Cmd+Shift+T`
- 润色：`Cmd+Shift+P`

## ⚠️ 注意事项

1. **避免系统快捷键冲突**
   - 不要使用系统保留的快捷键（如 `Ctrl+C`、`Ctrl+V` 等）
   - 避免与常用软件的快捷键冲突

2. **权限要求**
   - macOS 用户需要授予"辅助功能"权限
   - Windows 用户可能需要管理员权限

3. **快捷键生效**
   - 设置保存后立即生效
   - 无需重启程序

4. **跨平台差异**
   - Windows/Linux 主要使用 `Ctrl`
   - macOS 主要使用 `Cmd`
   - 配置会自动适配当前系统

## 🐛 常见问题

### Q: 快捷键不生效？
A: 
- 检查是否授予了辅助功能权限（macOS）
- 确认快捷键没有被其他程序占用
- 尝试更换其他快捷键组合

### Q: 如何恢复默认快捷键？
A:
- 删除 `config.json` 文件，程序会自动生成默认配置
- 或手动设置为 `Ctrl+Q` 和 `Ctrl+Shift+Q`

### Q: 可以使用纯字母作为快捷键吗？
A: 不可以。为了避免干扰正常输入，必须包含至少一个修饰键（Ctrl/Shift/Alt/Cmd）。

### Q: 支持鼠标按键作为快捷键吗？
A: 当前版本不支持，未来可能会添加。
